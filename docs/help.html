Template Box lets you store commonly-used phrases in a Google Spreadsheet, and copy them to clipboard easily.
You can organize them by tags for easier filtering.
  1. Sign in with your Google account. The only access we ask is to select a specific file and open it.
  2. Select a Google Sheet to work off. You can copy off this file to start.
    Note that the page URL changes to reflect the Google Sheet ID. You can return to that page later.
  3. Browse the templates and click the one you want. It'll be copied to your clipboard
